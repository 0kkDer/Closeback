# 👜 Closeback Totebag Project

Selamat datang di repositori resmi **Closeback Totebag**. Proyek ini didedikasikan untuk desain, katalog, dan dokumentasi produk totebag minimalis dengan fitur pengaman tambahan (*closeback*) seperti resleting atau *magnetic button*.

---

## 📌 Tentang Produk

Totebag konvensional sering kali mengorbankan keamanan demi estetika ringkas. **Closeback Totebag** hadir sebagai solusi modular yang menggabungkan kapasitas besar khas totebag dengan sistem penutup yang aman untuk melindungi barang bawaan harian Anda (laptop, buku, dompet).

### Fitur Utama:
* **Secure Closeback System:** Menggunakan *zipper* (resleting) berkualitas tinggi / kancing magnet tersembunyi.
* **Minimalist Design:** Estetika bersih, tanpa corak berlebih, cocok untuk kuliah, kerja, maupun kasual.
* **Kompartemen Laptop:** Dilengkapi dengan busa pelindung di bagian dalam (opsional).
* **Material Premium:** Menggunakan bahan Kanvas Drill / Cordura tahan air (*water-resistant*).
